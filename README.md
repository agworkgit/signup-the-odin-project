## The Odin Project: Sign-up Form

### Description

A responsive sign-up form with phone, email and password validation.

## User Story

AS A user that wants to register on a website

I WANT a easy signup form

SO THAT I can quickly access the information available on the website

## Acceptance Criteria

The app should:

Validate the inputs such as phone number, email address and the password

## Usage

The following image shows the web application's appearance and functionality:

![This web app is built with the use of pure HTML, CSS and JS.](/resources/images/demo.png)

> **Note**: This layout is designed to be responsive.

The following link will take you to the deployed webpage <https://agworkgit.github.io/signup-the-odin-project/>
